# RAJAYU
Simulating the dynamics of inflation with a modern touch

## Aim
To create a modern and state-of-the-art simulation of the early universe. The focus is on building a non-minimal inflation model. At the moment, we have successfully completed the first step of laying the foundation with a toy problem for a minimal model of chaotic inflation. In addition to this the focus is equally on performance, making use of the latest progress in the domain of HPC. 

## Building
Clone the repo using ```git clone https://github.com/aayush21-work/RAJAYU.git``` followed by running the ```make``` command. After the codes are compiled then execute the command ```./run```

## Dependencies
The compilation has been tested on the latest gcc compiler and should work on the earlier ones as well. Feel free to raise an issue if faced with any compilation errors or running the executables.
